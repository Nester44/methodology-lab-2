## Instruction
The project requires Python and Unittest to be installed on your machine.

Clone repository
```bash
git clone https://github.com/Nester44/methodology-lab-2.git
```

```bash
cd methodology-lab-2
```

Install dependencies:
```bash
npm install
```

To run tests:
```bash
npm run test
```
